# Eslint Plugin

This repo contains an eslint plugin that makes sure users of the generator adhering to the separate entrypoints.

## Usage

<Steps>

### Install dependencies

```bash
pnpm add -D @prisma-enums/eslint-plugin
```

### Add to your eslint config

```js
module.exports = {
  plugins: ["@prisma-enums"],
  rules: {
    "@prisma-enums/restrict-imports": "error",
  },
};
```

</Steps>

With the plugin activated, you will get an eslint error if you try to import from the wrong entrypoint.

```ts
// ✅ Allowed
import { Role } from "~/prisma/generated/enums";

// ❌ Not allowed
import { Role } from "@prisma/client";
```
