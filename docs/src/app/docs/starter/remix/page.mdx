# Remix Starter

<Steps>

### Install dependencies

```bash
pnpm add maple-sdk
```

### Initliaze Analytics Component

<Callout type='info'>Your find your website id in the dashboard settings</Callout>

```tsx
// app/root.tsx
import { Analytics } from 'maple-sdk/react'

export default function App() {
	return (
		<html lang='en'>
			<body>
				<Analytics token={'yourWebsiteID'} />
			</body>
		</html>
	)
}
```

### Deploy and wait for data to come in!

</Steps>
